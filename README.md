# 📝 Todo List App

A simple and clean Todo List Application built using **HTML, CSS, and JavaScript**.

This app helps users manage their daily tasks by allowing them to add and remove tasks easily.

## 🚀 Features

- ➕ Add new tasks
- ❌ Remove tasks
- 📋 Display task list
- 🎨 Clean and simple UI

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
